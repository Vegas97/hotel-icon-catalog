How to use any of these

▸ Iconify (CSS class) — swap : for --:   website mdi:home → code <span class="iconify mdi--home" />
▸ Iconify (React) — import { Icon } from "@iconify/react" → <Icon icon="tabler:door-enter" />
▸ Lucide — import { House } from "lucide-react" → <House className="size-4" />
▸ Radix — import { HomeIcon } from "@radix-ui/react-icons"
